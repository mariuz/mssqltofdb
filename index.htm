MSSQL And Access To InterBase Migration Wizard

Birthday Boy Edition

This document describes the MSSQL And Access To InterBase Migration Wizard (Wizard for short), and provides and introduction to the homepage on IBPhoenix.

The Wizard (also known as sql2gdb is a small application which helps migrate database from Microsoft SQL Server and Microsoft Access to InterBase. The application is Open Source, and for the moment is not protected under any license. You can copy it, distribute it, modify it, and do whatever you want with it, as long as you take full responsability.

As a historical note, the Access To InteBase Wizard (also known as mdb2gdb) was a previous version, which was distributed through the IBConversions e-group. The mdb2gdb wizard is no longer under development, because its functionality has been superceded by sql2gdb. If you still have a copy of mdb2gdb around, do yourself a favor and download this new Wizard.
